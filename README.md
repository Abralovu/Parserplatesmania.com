# PlatesMania Scraper

Парсер сайта platesmania.com — собирает фотографии номерных знаков с описанием.

## Что собирает

- Номерной знак
- Фотография (скачивается локально)
- Марка и модель автомобиля
- Регион
- Страна

Результат сохраняется в Excel файл с фотографиями внутри.

---

## Установка (один раз)

### 1. Установи Python 3.11+
Скачай с https://python.org и установи.

### 2. Скачай проект
```
git clone https://github.com/Abralovu/Parserplatesmania.com
cd Parserplatesmania.com
```

### 3. Создай виртуальное окружение
```
python3 -m venv venv
source venv/bin/activate        # Mac / Linux
venv\Scripts\activate           # Windows
```

### 4. Установи зависимости
```
pip install -r requirements.txt
playwright install chromium
```

### 5. Настрой конфиг
Скопируй файл `.env.example` и переименуй в `.env`:
```
cp .env.example .env
```

Открой `.env` и укажи путь к своему профилю Chrome:
```
CHROME_PROFILE=~/Library/Application Support/Google/Chrome/Default
```

Как найти свой профиль: открой Chrome → в адресной строке напиши `chrome://version/` → смотри строку "Путь к профилю".

## Использование

### Запуск парсинга
```
python main.py scrape --country ru --start 31000000 --end 31100000
```

- `--country` — код страны: `ru` Россия, `de` Германия, `us` США, `kz` Казахстан и т.д.
- `--start` — с какого ID начать
- `--end` — каким ID закончить
- `--fresh` — начать заново, игнорируя предыдущий прогресс

### Получить Excel с фотографиями
```
python main.py excel --country ru
```
Файл сохранится в папку `data/` — `plates_ru.xlsx`

### Получить CSV
```
python main.py export --country ru
```

### Посмотреть статус — сколько собрано
```
python main.py status --countries ru,de,us
```

### Сбросить прогресс и начать заново
```
python main.py reset --country ru
```

## Где хранятся данные
```
data/
├── plates.db          ← база данных со всеми записями
├── checkpoint.json    ← прогресс парсинга (не удалять!)
├── plates_ru.xlsx     ← Excel выгрузка
├── plates_ru.csv      ← CSV выгрузка
└── photos/
    └── ru/
        ├── 31000001.jpg
        ├── 31000002.jpg
        └── ...
```

**Важно:** папку `data/` не удалять — там все собранные данные и прогресс.

## Как получить старые данные

Все данные хранятся в `data/plates.db`. Чтобы выгрузить в Excel в любой момент:
```
python main.py excel --country ru
```

## Частые вопросы

**Парсер остановился — что делать?**
Просто запусти снова ту же команду без `--fresh`. Он продолжит с места остановки автоматически.

**Открывается окно браузера — это нормально?**
Да. Сайт использует защиту от ботов, поэтому парсер работает через реальный Chrome. Окно можно свернуть, не закрывать.

**Почему не все ID дают результат?**
Не все ID существуют на сайте — это нормально. Парсер автоматически пропускает пустые страницы.

**Где взять диапазон ID?**
Последние загруженные номера видны на главной странице сайта. Максимальный ID для России сейчас около 31 400 000. Для полного сбора запускай диапазонами по 100 000.

**Как парсить другую страну?**
```
python main.py scrape --country de --start 1 --end 500000
```
Коды стран: ru, de, us, kz, ua, by, uz, pl, fr, и другие — как в URL на сайте.
